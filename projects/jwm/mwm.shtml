This is a list of Motif Window Manager (MWM) hints that JWM supports.

Window Functions

Function Supported?
MWM_FUNC_ALL Yes
MWM_FUNC_RESIZE Yes
MWM_FUNC_MOVE Yes
MWM_FUNC_MINIMIZE Yes
MWM_FUNC_MAXIMIZE Yes
MWM_FUNC_CLOSE Yes

Window Decor

Decor Supported?
MWM_DECOR_ALL Yes
MWM_DECOR_BORDER Yes
MWM_DECOR_RESIZEH No
MWM_DECOR_TITLE Yes
MWM_DECOR_MINIMIZE Yes
MWM_DECOR_MAXIMIZE Yes

Other Hints

JWM does not support the MWM input hints nor does it support status hints.
