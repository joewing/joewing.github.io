These programs require the Ion shell to run. All of the files on this page include both the TI-83 and the TI-83 Plus versions of the programs. The TI-83 versions are named name.83g and the TI-83 Plus versions are named name.8xg. To send these programs to you calculator, unzip the archive, open the TI-Graph Link software, and select "Send To... RAM" from the "Link" menu and select the 8?g file to send. You must have both the graph link and the graph link software to use these programs. See TI's page for more information.
See the Ion Documentation for more information on Ion.

The source for these programs and more is available at GitHub.
